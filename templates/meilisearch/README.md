# Meilisearch

[Meilisearch](https://github.com/meilisearch/meilisearch) — Fast open-source search engine with typo-tolerant full-text search

## Quick Start

1. **Copy the environment file:**

   ```bash
   cp .env.example .env
   ```

2. **Start the service:**

   ```bash
   docker compose up -d
   ```

3. **Access the service:**

   Open [http://localhost:7700](http://localhost:7700) in your browser.

## Configuration

Copy `.env.example` to `.env` and edit the values as needed.

## Service Details

The docker-compose.yml exposes environment variables documented in `.env.example`.
