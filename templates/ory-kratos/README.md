# Ory Kratos

[Ory Kratos](https://github.com/ory/kratos) — self-hosted via Docker Compose.

## Quick Start

1. Copy the environment file and adjust as needed:

   ```bash
   cp .env.example .env
   ```

2. Start the service:

   ```bash
   docker compose up -d
   ```

3. Access the service:

   Open [http://localhost:4433](http://localhost:4433) in your browser.

## Configuration

Edit `.env` to customize port and other settings. See `.env.example` for available options.
